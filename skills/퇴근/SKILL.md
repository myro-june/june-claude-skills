---

네이버 웍스 퇴근 체크를 실행합니다.

1. `bash "${CLAUDE_PLUGIN_ROOT}/scripts/naver-works-checkout.sh"` 를 실행한다.
2. 결과를 확인하고 사용자에게 알려준다. 결과에 `|CHECKIN:HH:MM`이 포함되어 있으면 출근 시간을 파싱하여 총 근무 시간을 계산한다.
   - `SUCCESS....|CHECKIN:HH:MM` → 현재 한국 시간(KST)에서 출근 시간을 빼서 총 근무 시간을 계산한다. "퇴근 처리 완료되었습니다! 오늘 총 N시간 M분 근무하셨습니다. 수고하셨습니다."
   - `SUCCESS` (CHECKIN 없음) → "퇴근 처리 완료되었습니다! 오늘도 수고하셨습니다."
   - `ALREADY: HH:MM|CHECKIN:HH:MM` → 출근~퇴근 시간으로 총 근무 시간을 계산한다. "이미 퇴근 상태입니다. HH:MM에 퇴근하셨습니다. (오늘 총 N시간 M분 근무)"
   - `ALREADY: HH:MM` (CHECKIN 없음) → "이미 퇴근 상태입니다. HH:MM에 퇴근하셨습니다."
   - `FAIL_NOT_CHECKEDIN` → "아직 출근 처리가 되지 않은 상태입니다. 먼저 출근 체크를 해주세요."
   - `FAIL_UNVERIFIED` → 퇴근 버튼은 클릭했지만 상태 변경이 확인되지 않은 경우. `|CHECKIN:HH:MM`이 포함되어 있으면 출근 시간도 함께 알려준다. 에러 메시지를 그대로 사용자에게 전달하고, 다음을 안내한다:
     > 네이버 웍스에서 직접 퇴근 상태를 확인해주세요.
     > 만약 반복적으로 이 문제가 발생한다면:
     > 1. Chrome 주소창 오른쪽의 팝업 차단 아이콘을 클릭 → "항상 허용" 선택
     > 2. 또는 Chrome 설정 → 개인정보 및 보안 → 사이트 설정 → 팝업 및 리디렉션 → `home.worksmobile.com` 허용 추가
   - `FAIL` → 에러 내용을 분석하고 사용자에게 상황을 설명한다. 스크립트 실행 오류인 경우 Chrome 실행 여부, macOS 접근성/자동화 권한 설정을 확인하도록 안내한다.
