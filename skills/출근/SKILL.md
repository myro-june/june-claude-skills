네이버 웍스 출근 체크를 실행합니다.

1. `bash "${CLAUDE_PLUGIN_ROOT}/scripts/naver-works-checkin.sh"` 를 실행한다.
2. 결과를 확인하고 사용자에게 알려준다.
   - `ALREADY: HH:MM` → 현재 한국 시간(KST)과 출근 시간의 차이를 계산하여 "이미 출근 상태입니다. HH:MM에 출근하셨습니다. (현재 N시간 M분 근무 중)" 형식으로 알려준다.
   - `ALREADY: (시간 확인 불가` → "이미 출근 상태입니다."
   - `SUCCESS` → "출근 처리 완료되었습니다!"
   - `FAIL_UNVERIFIED` → 출근 버튼은 클릭했지만 상태 변경이 확인되지 않은 경우. 에러 메시지를 그대로 사용자에게 전달하고, 다음을 안내한다:
     > 네이버 웍스에서 직접 출근 상태를 확인해주세요.
     > 만약 반복적으로 이 문제가 발생한다면:
     > 1. Chrome 주소창 오른쪽의 팝업 차단 아이콘을 클릭 → "항상 허용" 선택
     > 2. 또는 Chrome 설정 → 개인정보 및 보안 → 사이트 설정 → 팝업 및 리디렉션 → `home.worksmobile.com` 허용 추가
   - `FAIL` → 에러 내용을 분석하고 사용자에게 상황을 설명한다. 스크립트 실행 오류인 경우 Chrome 실행 여부, macOS 접근성/자동화 권한 설정을 확인하도록 안내한다.
