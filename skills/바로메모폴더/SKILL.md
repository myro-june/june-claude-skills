# /바로메모폴더 — 메모.app 폴더 확인 및 추가

메모.app의 현재 폴더 구성을 확인하고, 필요한 폴더를 추가합니다.

## When to Activate

사용자가 명시적으로 `/바로메모폴더`를 입력했을 때만 실행.

## Workflow

### Step 1: 현재 폴더 목록 조회

```bash
osascript -e 'tell application "Notes" to get name of every folder'
```

결과를 목록으로 보여준다:
```
현재 메모.app 폴더:
- 001-하루 인박스
- 002-주간 인박스
- 101-자주 상기
- 102-개발
- ...
```

## 폴더 네이밍 컨벤션

- `001`, `002` — 인박스 계열 (하루/주간)
- `101`, `102`, `103`, `104`, `105`, `106` ... — 카테고리 폴더 (자주 상기, 개발, 회사, 개인, 아카이브, 잡념 등)
- 새 폴더 추가 시 카테고리는 `10x` 번대의 다음 번호를 사용한다.

### Step 2: 폴더 추가

**인자가 있는 경우** (`/바로메모폴더 107-프로젝트`):
- 해당 폴더가 이미 존재하는지 확인한다.
- 존재하면: "이미 존재합니다: {폴더명}" 알림
- 없으면: 폴더를 생성한다.

**인자가 없는 경우** (`/바로메모폴더`):
- 현재 폴더 목록만 보여주고 종료한다.
- 사용자가 추가를 원하면 폴더명을 입력받는다.

### Step 3: 폴더 생성

```bash
osascript -e 'tell application "Notes" to make new folder with properties {name:"FOLDER_NAME"}'
```

### Step 4: 결과 확인

- 성공 시: "폴더를 추가했습니다: {폴더명}" 한 줄로 알림
- 실패 시: 에러 내용을 설명
